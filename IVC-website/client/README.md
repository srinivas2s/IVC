# React + Vite 

A fast and modern React setup powered by **Vite**, providing instant Hot Module Replacement (HMR), optimized builds, and a clean development experience.

---

## Overview

This project uses **React** with **Vite** as the build tool, offering:
- Lightning-fast development server
- Instant Hot Module Replacement (HMR)
- Optimized production builds
- Minimal and scalable project structure
